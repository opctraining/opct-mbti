# opct_mbti
나의 운동취향을 알아보는 mbti

## How to contribute?
### Preliminary
1. user configuration을 repository 내의 `.git/config`에 등록

    ```bash
    git config --local user.email {your email}
    git config --local user.name {your name or your nick}
    ```

2. 아래의 command를 실행

    ```bash
    make all
    ```
